Project Information

This website presents preliminary data on U.S. National Parks that are part of the Central Grassland Initiative. Parks were selected based on their location entirely within the Central Grasslands Roadmap boundary (see source information below). Data sources included in this dashboard include:

  • Central Grasslands Roadmap Assessment Map
  • Administrative boundaries of NPS units downloaded 7/24/2025: IRMA Record 2314463
  • Park visitation data for 2024: NPS Visitation Query Builder
Data were compiled by Kate Miller in July 2024. All code can be accessed on the following GitHub repository: katemmiller/centralGrasslands
